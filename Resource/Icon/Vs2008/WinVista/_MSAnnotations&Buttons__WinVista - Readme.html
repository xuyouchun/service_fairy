Visual Studio 图像库：批注和按钮 (WinVista)

图标图片
此文件类型可以包含多个图像大小和/或颜色深度，并且通常在可提供多种查看样式的控件中使用。例如，允许用户在“小图标视图”和“大图标视图”之间进行选择的列表框就使用了这种文件。使用这种文件的控件会显示适当的图像大小。请注意，最好是通过文件系统来查看这些图像；下表中的外观可用来标识图像，但这仅是多种表示形式中的一种。

使用限制
作为可视语言的一部分，下列图像（或图像的任意部分）的使用方式必须与下面的说明一致：

图像 文件名 用法
error.ico 一般错误
Gear.ico 表示某些要求进行服务器端处理的内容的一般图像
info.ico 表示的信息
Likely_unavail.ico 某些内容是不可用的
New.ico 与某些新内容或刚刚创建的内容相关
preferences.ico 首选项
share_overlay.ico 用作覆盖以表示标记对象可共享使用
Shortcut.ico 用作覆盖以表示对象是指向源文档或文件的快捷方式（链接）。
UAC_shield.ico 用户帐户控制。有关使用要求，请参阅 MSDN 文章。
warning.ico 一般警告图标
