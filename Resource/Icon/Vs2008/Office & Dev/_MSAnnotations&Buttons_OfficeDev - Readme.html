Visual Studio 图像库：批注和按钮（Office，Dev）

图标图片
此文件类型可以包含多个图像大小和/或颜色深度，并且通常在可提供多种查看样式的控件中使用。例如，允许用户在“小图标视图”和“大图标视图”之间进行选择的列表框就使用了这种文件。使用这种文件的控件会显示适当的图像大小。

使用限制
作为可视语言的一部分，下列图像（或图像的任意部分）的使用方式必须与下面的说明一致：

图像 文件名 用法
error.ico 一般错误符号
eventlogError.ico 事件日志中的错误
eventlogInfo.ico 表示事件日志中的有用信息
eventlogWarn.ico 表示事件日志中的警告
